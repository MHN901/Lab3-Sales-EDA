# Lab 3: Exploratory Data Analysis (EDA) - Sales Analysis

## Overview
This repository contains the solution for Lab 3 of the Machine Learning course (ARTI308). The goal of this assignment was to perform Exploratory Data Analysis (EDA) on a dataset to uncover trends and insights.

## Dataset
**Name:** Superstore Sales Dataset
**Source:** Kaggle
**Description:** The dataset contains retail sales data including columns for:
- Order Date
- Sales Amount
- Profit
- Region
- Product Category

## Key Findings
Based on the analysis performed in the notebook:
1.  **Top Selling Category:** Technology generates the highest revenue.
2.  **Most Profitable Region:** The West region is the most profitable.
3.  **Sales Trends:** Sales tend to increase towards the end of the year (Seasonal trend).

## Files in this Repository
* `Lab3_Assignment.ipynb`: The Jupyter Notebook containing the Python code and analysis.
* `Superstore.csv`: The dataset used for analysis.

## Tools Used
* Python
* Pandas (for data manipulation)
